# UKNC FPGA

Мы рады представить UKNC FPGA.

Автор - Алексей Гуров.

Вам потребуется:
- tang20nano
- bl616 board
- свежий PC с USB интерфейсом

## ОБРАТИТЕ ВНИМАНИЕ

Алексей поменял распиновку выводов относительно
исходного Atari репозитория!!!

```
tang20nano    bl616
71            io12
72            io13
73            io10
74            io11
75            io14
GND           GND
+5            +5
```
